# rudderstack_destination_snowflake (Resource)

This resource represents a Snowflake destination. For more information check 
https://www.rudderstack.com/docs/destinations/warehouse-destinations/snowflake/

## Example Usage

```terraform
resource "rudderstack_destination_snowflake" example{
  name = "my-snowflake"

  config {
    account = "..."
    database = "..."
    warehouse = "..."
    user = "..."
    password = "..."
    namespace = "..."
    sync {
      frequency = "60"
      # start_at                  = "10:00"
      # exclude_window_start_time = "11:00"
      # exclude_window_end_time   = "12:00"
    }
    # json_paths = "..."
    use_rudder_storage = true
    # cloud_provider = "AWS"
    # additional_properties = true
    # s3 {
    #   bucket_name = "..."
    #   access_key_id = "..."
    #   access_key = "..."
    #   enable_sse = true
    # }
    # gcp {
    #   bucket_name = "..."
    #   credentials = "..."
    #   storage_integration = "..."
    # }
    # azure {
    #   container_name = "..."
    #   account_name = "..."
    #   account_key = "..."
    #   storage_integration = "..."
    # }
    # prefix = "..."
  }
}
```
