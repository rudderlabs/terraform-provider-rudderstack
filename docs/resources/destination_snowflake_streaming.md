# rudderstack_destination_snowflake_streaming (Resource)

This resource represents the Snowflake Streaming destination. For more information check
https://www.rudderstack.com/docs/destinations/streaming-destinations/snowflake-streaming/

## Example Usage

```terraform
resource "rudderstack_destination_snowflake_streaming" "example" {
  name = "my-snowflake-streaming"

  config {
    account   = "your_snowflake_account"
    database  = "your_database"
    warehouse = "your_warehouse"
    user      = "your_user"
    private_key = <<EOT
-----BEGIN PRIVATE KEY-----
YOUR_PRIVATE_KEY
-----END PRIVATE KEY-----
EOT
    namespace = "your_namespace"
  }
}
```
