# rudderstack_retl_source_model (Resource)

A RETL (Reverse ETL) source backed by a SQL model. The provided SQL query is executed periodically against the warehouse identified by `account_id`, and the result set is synced to the connected destinations.

## Example Usage

```terraform
resource "rudderstack_retl_source_model" "users_revenue" {
  name                   = "users-revenue"
  source_definition_name = "snowflake"
  account_id             = "acc-1234"

  config {
    primary_key = "user_id"
    sql         = "select user_id, sum(amount) as revenue from orders group by user_id"
    description = "Revenue per user, refreshed nightly."
  }
}
```
