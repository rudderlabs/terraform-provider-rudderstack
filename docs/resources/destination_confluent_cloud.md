# rudderstack_destination_confluent_cloud (Resource)

This resource represents a Confluent Cloud destination. For more information check
https://www.rudderstack.com/docs/destinations/streaming-destinations/confluent-cloud/

## Example Usage

```terraform
resource "rudderstack_destination_confluent_cloud" "example" {
  name = "my-confluent-cloud"

  config {
    bootstrap_server = "test.region.provider.confluent.cloud:9092"
    topic            = "example-topic"
    api_key          = "example-api-key"
    api_secret       = "example-api-secret"
  }
}
```
