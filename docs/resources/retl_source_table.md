# rudderstack_retl_source_table (Resource)

A RETL (Reverse ETL) source backed by a single warehouse table. RudderStack syncs rows from the configured `schema.table` to the connected destinations, using `primary_key` to track changes.

## Example Usage

```terraform
resource "rudderstack_retl_source_table" "users" {
  name                   = "users"
  source_definition_name = "snowflake"
  account_id             = "acc-1234"

  config {
    primary_key = "id"
    schema      = "public"
    table       = "users"
  }
}
```
